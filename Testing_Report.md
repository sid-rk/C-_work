#### 6.2 发现并修复的问题

在测试过程中发现并解决了以下问题：

1. **下一曲不循环**：Sequential 模式下播到最后一首按下一曲直接停止。修复方案：修改 `PlayNext()` 方法，所有模式下超出队列末尾时回到索引 0。
2. **暂停失效**：停止状态调用 `TogglePlayPause()` 会重新触发播放。修复方案：添加 `_isPaused` 状态变量，严格区分暂停与停止两个状态。
3. **进度条闪烁**：视频模式下音频定时器仍在更新进度条。修复方案：进入视频模式时停用音频定时器，退出时恢复。

#### 6.3 TDD 实践总结

本项目开发中实践了部分 TDD 理念：

- 在编写 FFT 频谱分析功能前，先确定输入（PCM 样本数组）和输出（幅值数组）的接口格式，再实现具体的 `ComputeFftFromSamples` 方法
- 数据库 CRUD 操作在编写 UI 前先通过控制台测试 SQL 语句的正确性
- 播放模式逻辑的边界条件（最后一首、空列表、单曲循环）在实现前就列出测试用例

AI 辅助工具（Claude Code）在测试过程中帮助自动生成测试数据、模拟边界条件，以及对异常捕获代码进行安全性审查。
